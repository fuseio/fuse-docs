## Pocket Network

[Pocket Network](https://www.pokt.network/) is a decentralized RPC infrastructure provider with 52,000+ nodes across 25+ countries, offering true decentralization for blockchain access. Pocket provides a cost-effective alternative to centralized providers like Infura, Alchemy, and QuickNode, with 70–80% cost reduction through a per-million-requests pricing model via staked token burn.

**Key highlights:**

* 70–80% cost savings compared to centralized RPC providers
* Truly decentralized: 52,000+ nodes in 25+ countries
* Pricing based on per million requests via staked token burn
* Free public RPC access available

To get started, visit [Pocket Network Public RPC Support](https://www.pokt.network/support-public-rpc) or learn more about [Foundation Partnerships](https://www.pokt.network/foundation-partnerships).
