# Introduction

The FuseBox API allows you to develop and integrate your applications seamlessly. With the Fuse API, you have access to the Production environment where your applications are deployed and made available to your customers. So, whether you are developing a new application or integrating an existing one, the Fuse API gives you the flexibility and control you need to deliver high-quality applications with ease.

## Production

The Production environment is meant for live use and real transactions. It is recommended to use this environment once you have finished testing and are ready to go live. The base URL for the Production environment is [https://api.fuse.io/api](https://api.fuse.io/api).

Please note, all API calls should be made over HTTPS.
