# Bridge to Fuse

Currently, the following cross-chain bridges enable the transfers of digital assets between Fuse and other major blockchains.

## Fuse Bridge

It can be accessed via the Fuse Tool called [Console](https://console.fuse.io/bridge). Console contains a Bridge which allows users to move funds from different networks and centralized exchanges to Fuse. The Bridge connects Fuse Network to **Ethereum**, **Polygon**, **Arbitrum**, **Optimism**, **Base**, **BNB Chain** and **Solana**. It supports the transfers of significant assets, including **FUSE**, **USDC**, **USDT**, and **WETH**.

It also enables Bridging assets from Centralized Exchanges such as: Binance, Coinbase, KuCoin and Huobi Global.

## AllBridge

[Allbridge](https://app.allbridge.io/bridge?from=SOL&to=FUSE&asset=USDC) is an innovative cross-chain bridging solution designed to make asset transfers between blockchains safe, reliable and modern.
It offers native and wrapped token transfers between Ethereum Virtual Machine (EVM) and non-EVM compatible blockchains.

Use [Allbridge](https://app.allbridge.io/bridge?from=SOL&to=FUSE&asset=USDC) to effortlessly transfer funds from Solana to the Fuse Network blockchain.

## LayerSwap

[Layerswap](https://layerswap.io/app) enables instant and seamless asset bridging across centralized exchanges, blockchains, and wallets with minimal fees, significant gas savings, and excellent customer support.

Use Layerswap to effortlessly transfer funds from centralized exchanges to the Fuse Network blockchain.
