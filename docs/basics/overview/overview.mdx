# Fuse Network: Empowering Web3 Payments

Fuse Network is an EVM-compatible, Layer-1 public blockchain designed to simplify Web3 payments and make blockchain technology more accessible.

## Key Features

1. Layer-1 Blockchain: Independent, with finality achieved by Fuse Network validators.
2. EVM Compatibility: Supports Ethereum-based smart contracts and dApps without modification.
3. Public Network: Open for anyone to interact, deploy, or run nodes without permission.
4. Web3 Payments Focus: Enables efficient cryptocurrency transactions for businesses and users.
5. Account Abstraction: Simplifies onboarding and interaction for non-technical users.
6. Low-Cost Transactions: Fees as low as $0.0001 (10,000 transactions for $1).
7. Fast Confirmations: ~5 second block intervals.
8. High Throughput: ~180 native token transfers per second.
9. Interoperability: Compatible with Ethereum, Polygon, Arbitrum, Optimism, and BNB Chain.

## Consensus and Governance

- Uses delegated Proof of Stake (dPoS)
- Secured by independent validators
- Network changes adopted via Fuse Improvement Proposals (FIPs)

## FUSE Token

- Native utility token for gas fees and ecosystem incentives
- New tokens issued with each block

## Target Audience

- Crypto projects
- Startups
- Established companies
- Developers building Web3 payment solutions, gaming, or social applications
