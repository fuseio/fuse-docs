# Fuse Explorer

[Fuse Explorer](https://explorer.fuse.io/) is a full-featured block explorer for the Fuse Network and many other chains. Fuse Explorer is open-source software and is based on the Blockscout explorer.

About Blockscout:

> Blockscout is optimized for next-level blockchain exploration. Multichain, highly configurable, and open source, the explorer portal opens up access to all chain data and functionality. Search blocks, transactions and addresses, then go beyond to connect and interact with apps, bridges, wallets and more. The blockchain universe is yours to explore!

## Main Features

* Track Blocks, Accounts, Balances, and Token transfers
* API-based access to blockchain information
* The ability to read and authenticate smart contracts
* Easily tag and keep tabs on your transactions by using the tools under "My Account."

### Blockscout Pro API

Blockscout recently launched their Pro API — a premium blockchain data service that provides enhanced access to on-chain data across multiple networks, including Fuse. It offers higher rate limits, custom endpoints, and improved performance compared to the standard explorer API.

**Key highlights:**

* Higher rate limits and improved performance for production workloads
* Multichain support through unified endpoints using chain IDs
* Multiple API types: REST API v2, JSON RPC API v2 (Etherscan-compatible), and ETH RPC API
* Credit-based usage system with a free tier available

To get started, visit the [Blockscout Developer Portal](https://dev.blockscout.com/) or read the [Pro API documentation](https://docs.blockscout.com/devs/pro-api).
