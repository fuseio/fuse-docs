## Connect to Ember

Add Ember to your wallet or development environment to get started.

Use the information below to connect and submit transactions to Ember.

## Testnet

| Property        | Description                                |
| --------------- | ------------------------------------------ |
| Name            | Flash Testnet                              |
| Description     | The public testnet for Ember.              |
| Chain ID        | 10920                                      |
| RPC URL         | https://rpc.flash.fuse.io                  |
| Explorer URL    | https://fuse-flash.explorer.quicknode.com/ |
| Sequencer URL   | https://sequencer.flash.fuse.io            |
| Currency Symbol | ETH                                        |

<ConnectButton />

Once connected, use a [bridge](https://fuse-flash.bridge.quicknode.com/) or [faucet](https://faucet.quicknode.com/fuse/flash) to get testnet funds into your wallet.
